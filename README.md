The official DogeHouse.js bot &amp; for DogeHouse.tv.


# Self host :  


```bash
git clone https://github.com/dogegarden/gardener
cd gardener-master
npm i 
```
Edit the .env and fill in your tokens 
Edit index.js file and fill in your room ID

```bash
node index.js
```

See the magic happen !
